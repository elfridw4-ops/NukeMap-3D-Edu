# Variables d'Environnement - NUKEMAP EDU

Toutes les variables sont optionnelles pour le fonctionnement de base du simulateur, mais peuvent être requises pour des extensions futures.

| Variable | Utilité | Obligatoire |
|----------|---------|-------------|
| `VITE_MAPBOX_TOKEN` | (Si bascule vers Mapbox) | Non |
| `VITE_APP_VERSION` | Utilisé pour le suivi des versions | Non |
