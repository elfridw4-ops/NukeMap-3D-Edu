# Limitations Connues - NUKEMAP EDU

## Technique
- **Précision Géographique** : Les obstacles de relief (montagnes) ne sont pas pris en compte pour l'onde de choc (modèle 2D étendu en 3D).
- **iOS Safari** : L'installation PWA peut être moins intuitive (nécessite le bouton "Partager" -> "Sur l'écran d'accueil").

## Physique
- La retombée radioactive (fallout) est une approximation basée sur la direction constante du vent.
