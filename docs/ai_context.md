# Contexte IA - NUKEMAP EDU

## Choix de Conception
- **Antigravity Agent** : Utilisé pour la génération de code robuste et l'architecture système.
- **Modèles physiques** : Privilégier les calculs mathématiques explicites dans le code (`nuclearMath.ts`) plutôt que des approximations générées.

## Hypothèses
- L'utilisateur dispose d'un navigateur récent avec support WebGL 2.
- La précision des données bathymétriques OSM est suffisante pour une estimation pédagogique des tsunamis.

## Stratégie UX
- Utiliser la landing page pour filtrage pédagogique avant d'exposer la complexité de la console de simulation.
