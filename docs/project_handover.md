# Transfert de Projet (Handover) - NUKEMAP EDU

## Résumé
NUKEMAP EDU est un simulateur nucléaire 3D robuste conçu pour la résilience. L'architecture repose sur React et Deck.gl avec une forte emphase sur la persistance locale des données.

## Points d'attention
- Le fichier `nuclearMath.ts` contient la logique métier critique.
- `App.tsx` centralise l'état persistant via `hooks.ts`.

## État actuel
Le projet est en phase de finalisation PWA avec une documentation complète en place.
