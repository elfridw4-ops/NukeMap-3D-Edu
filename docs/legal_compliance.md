# Conformité Juridique - NUKEMAP EDU

## Données Collectées
- Aucune donnée personnelle n'est collectée ou stockée sur un serveur.
- Les paramètres de simulation sont stockés localement sur le terminal de l'utilisateur (`localStorage`).

## Localisation
- Cible principalement les pays francophones pour l'éducation aux risques nucléaires.
- Conformité RGPD : Pas de collecte de données, aucun consentement requis pour les cookies car purement fonctionnels (stockage local de préférence technique).

## Avertissement (Disclaimer)
L'application est à but purement pédagogique. Les résultats ne doivent pas être utilisés pour des prévisions de sécurité réelle ou des plans d'urgence officielle.
